# Talking Rabbitt – Conversational Business Intelligence

Talking Rabbitt is an AI-powered conversational analytics tool that allows business leaders to interact with their data using natural language.

## Features

- Upload sales CSV
- Ask natural language questions
- AI returns insights
- Automatic chart visualization

Example question:

"Which region had the highest revenue?"

## Tech Stack

Streamlit
Python
Pandas
Matplotlib

## How to Run

pip install -r requirements.txt
streamlit run app.py
